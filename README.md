# Intent Example — Busca no Google Maps
 
Aplicação Android em Kotlin que permite ao usuário digitar um termo de busca, como "Restaurante" ou "Farmácia", e encontrar estabelecimentos próximos à sua localização atual.
 
Ao clicar no botão de busca, o app solicita permissão de localização, obtém as coordenadas do dispositivo e abre o Google Maps com os resultados filtrados para a região do usuário.
